## Default Permission

Default permissions for the Windows taskbar lyric plugin

#### This default permission set includes the following:

- `allow-close-taskbar-lyric`
- `allow-open-taskbar-lyric`
- `allow-open-taskbar-lyric-devtools`
- `allow-set-click-interception`
- `allow-set-forwarding-enabled`
- `allow-stop-mouse-hook`

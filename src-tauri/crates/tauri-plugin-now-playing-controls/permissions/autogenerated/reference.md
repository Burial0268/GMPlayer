## Default Permission

Default permissions for GMPlayer desktop now playing controls

#### This default permission set includes the following:

- `allow-clear`
- `allow-initialize`
- `allow-set-enabled`
- `allow-update-play-mode`
- `allow-update-state`
- `allow-update-timeline`

## Default Permission

Default permissions for the orientation plugin

#### This default permission set includes the following:

- `allow-set-screen-orientation`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`orientation:allow-set-screen-orientation`

</td>
<td>

Enables the setOrientation command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`orientation:deny-set-screen-orientation`

</td>
<td>

Denies the setOrientation command without any pre-configured scope.

</td>
</tr>
</table>
